### Objective : 
A framework designed to facilitate the development of automation test scripts for API services using Java.

### Tools / libraries used :

1. Java
2. Rest Assured
3. JUnit
4. Maven
5. Java Faker

### How to run :

1. Clone the repository
2. mvn clean install
3. View results
4. Run testng.xml
5. Go to > reports > current date > report.html (See below)
